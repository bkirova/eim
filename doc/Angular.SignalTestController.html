oscMessageReceivedListener(data) → {undefined}

This method is called when the server fires the oscMessageReceived event on the socket.io socket to which the client is connected.

If the message is a signal quality message, $scope#edaQuality or $scope#poxQuality is updated accordingly. If the message indicates that the test recording is complete, $scope#testRecordingComplete is updated accordingly and sendStopSignalTestMessage is called.
